JS High-level overview

High level

In any other low level programming language you have to manually manage the memory resources. for example you have to ask the computer for memory space to create a new Variable.

However with High level programming languages like JavaScript or Python we don't have to manage resources at all because of the so called abstractions in these languages, it takes all that work away from us.

The downside of these High level programming languages is that our applications wont be as optimized and fast as the applications made with Low level languages

Garbage collected

One of the powerful tools that takes away memory management from us developers is Garbage Collection.

It's an algorithm that runs in JavaScript engine, which then removes the old and unused Objects from the computer memory. This then allows us optimize our program automatically.

Multi paradigm

Paradigm is an approach and mindset of Structuring code. In short its our Style and Technique of writing code.

It's an algorithm that runs in JavaScript engine, which then removes the old and unused Objects from the computer memory. This then allows us optimize our program automatically.

There are 3 types of Paradigm in JavaScript:

  • procedural programming
  • Object-oriented programming (OOP)
  • Functional programming (FP)

procedural programming is Structuring our code in a linear fashion with some functions in between.

OOP programming will be explained later. for now we must know that OOP in JS is prototype-based. it means objects inherit methods from a prototype-based object and that prototyped object is like a blueprint.

Functional programming will be explained later.

first-class functions

In a language with first-class functions,functions are simply treated as variables. We can pass them into other functions, and return them from functions.

Dynamic-typed

In JavaScript we don' define the data type. types becomes known at runtime. and also type of variables automatically changes whenever we assign it a different data type.
