JS Engine and Runtime

JS Engine

It's a software that executes JavaScript code. Every engine consist of two parts:

  • Call stack
  • heap

call stack is where our code is executed by using something called execution context.

heap is an unstructured memory container where all objects in our code is saved.

Interpretation vs Compilation

In every programming language, codes need to be compiled into machine language. this conversion can happen either by compilation or interpretation method.
